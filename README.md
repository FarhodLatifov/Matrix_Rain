# Matrix Rain

Реализация эффекта «цифрового дождя» из фильма The Matrix на HTML, CSS и JavaScript.

## Демонстрация
Откройте `index.html` в браузере.

## Структура проекта
```

index.html   – разметка
style.css    – стили
main.js      – логика анимации

````

## Установка и запуск
```sh
git clone https://github.com/FarhodLatifov/Matrix_Rain.git
cd Matrix_Rain
````

Откройте `index.html` вручную в браузере.

## Механика

Скрипт генерирует столбцы случайных символов и анимирует их вертикальное падение. Анимация реализована через циклический рендер кадров в `main.js`.

## Кастомизация

В `main.js` можно изменить:

* скорость падения
* набор символов
* плотность столбцов

## Требования

Современный браузер с поддержкой HTML5.
